How to do this?

We just add rules to ".justreq" as follow, and code some files as upload.jrs

{
  ...
  "rules": [
    {
      "url":  "api/upload.do",
      "subs": "jrs/upload.jrs"
    }
  ]
}

These jrs files of this example are located in ".jr/subs/jrs/"
